# Complete Bone Meal
Make Bone Meal applicable to *all* growables!

The new "Bone Meal"-able crops include:
- Cactus
- Chorus Flowers
- Coral Blocks
- Corals
- Coral Fans
- Coral Wall Fans
- Dead Bushes
- Flowers
- Hanging Roots
- Leaves
- Lily Pads
- Mycelium
- Nether Wart
- Roots
- Sponges
- Spore Blossoms
- Sprouts
- Sugar Cane
- Tall Plants
- Vines

The mod also has a config to toggle functionality for any individual growable.
